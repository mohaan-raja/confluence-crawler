Title: Admin Experience - User Stories

Decisions Taken

  • An Organization can contain only Sales360 accounts and nothing else [Beta]
  • Only new customers
    • Upgrades from existing Freshsales customers is not part of the beta scope
  • Custom roles are not part of the scope for beta
    • Eventually, custom roles can be defined in GA
  • Omnibar functionality will not be available for Sales360 organizations.
