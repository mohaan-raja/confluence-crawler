Working with Generic email IDs

  • If Enrichment API failed to extract company, alternatives are :
    • If Twitter Handle is available
      • Lookup Full Contact with Twitter Handle
      • Use Twitter's API to Parse tweets for any highly retweeted pages/handles
    • If Facebook Handle is available
      • Use Facebook's Graph API to get person's occupation/education details
      • Lookup Full Contact with Facebook Handle/URL
    • If LinkedIn Handle is available
      • Use LinkedIn API to get person's current company and lookup company with Enrichment API
    • Lookup LinkedIn handle using email ID
