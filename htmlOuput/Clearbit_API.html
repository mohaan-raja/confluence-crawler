Title: Clearbit API

  • Enrichment API (Docs)
  • Requires email ID
  • Can be integrated with Salesforce.
  • Reveal API Tried by Marketing team (Jithin Isaac) 
  • Apparently still used by Freshchat (Mahesh S)

Personal Data (based on username in email ID) :

  • Social :
    • Github - Has bio, followers and following
    • Twitter - Has bio, followers, following and location 
    • Googleplus - Only handle
    • Linkedin - Only handle
    • facebook- Only handle
  • Location :
    • geo/Location - Has lat/long, City and Country 
    • utcOffset/timeZone
  • About : 
    • site
    • Bio - Might describe role if employment is NA
    • Name - Full Name 
    • Employment - Has role, seniority and title
  • Other : 
    • id
    • gravatar
    • fuzzy
    • indexedAt
    • emailProvider
    • email
    • Avatar

Company Data (based on username in email ID) : 

  • Social : 
    • Facebook - Has handle and likes
    • Twitter - Bio, followers, following and location
    • Linkedin - Has handle
    • Crunchbase - Has handle
  • Details/Stats : 
    • Type - Public/Private
    • Category - Sector and Industry
    • foundedYear
    • Metrics - Has No. of employee, Estimated ARR, Amount raised
  • About : 
    • Bio/Description
    • name/legalName/ticker
    • logo
    • Geo/location + utcOffset/timeZone - Info about where company’s office/ where it’s registered (San Bruno for Freshworks)
    • Tech - Some APIs used by the company
    • Site/Domain/domainAliases - Could help with dedupe ?
    • Tags - [u 'SAAS', u 'B2B', u 'Technology', u 'Information Technology & Services', u 'Enterprise'] for Freshworks
  • Other :
    • id
    • emailProvider
    • identifiers
    • Parent - Domain of parent company, if any
    • indexedAt
    • phone
    • facebook
