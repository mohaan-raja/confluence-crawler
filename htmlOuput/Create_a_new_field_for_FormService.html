Title: Create a new field for FormService

This guide is to help you create a new field for an entity which you want added into FormServ

Step-by-step guide

  1. Create DB migrations for your field and complete the migration.
  2. Create methods to add the new field in  Utils::FormServiceFieldAdd. Use the existing methods as reference.
  3. Add the field in FsFilter::Constants in the required arrays.
  4. Add the field in FormService::FormJsonBuilder. Add the field in the section in which you want it to be.
  5. Add the field in the required app/serializers (listing serializers also) as well as frontend/app/models and app/models as required.
  6. Add field for the model in Search::EsScrollReindex, Search::EsModelMapping and Search::EsTrackModel
  7. Create a new rake task in system_field.rake which call the method you created in Step 2
  8. Add translations and make changes to db/dummy_seeds and db/fixtures as required
