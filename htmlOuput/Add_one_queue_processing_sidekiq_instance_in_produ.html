Title: Add one queue processing sidekiq instance in production

Add a instance in sidekiq layer with name of queuename-sidekiq-#. This will process only that queue alone in all the processes.

Instance size - We are creating no.of.cpu-1 sidekiq processes with 10 threads in each process.

Ex: queue name - workflow_automation, then instance should be workflow_automation-sidekiq-1
