Title: Crunchbase API

  • Crunchbase for Salesforce : Enriches specific info about a company

API Overview

  • Access to the Full API Requires an Enterprise or Applications License
  • API Docs : https://data.crunchbase.com/docs/using-the-api
  • Accessing Crunchabse API and Open Data Map Rapid API (free plan requires credit card details too) : https://rapidapi.com/crunchbase/api/crunchbase/pricing
  • Crunchbase for Applications Deck 
  • About the data
    • Obtained though multiple sources. (Crawling, Crowdsourced, etc.)
    • Doesn't provide info about salesperson (from email ID). Only company properties. 
    • Some companies use a FullContact/Clearbit (besides Crunchbase) to enrich person info
    • Enriches Org category well (special mention)
    • Alexa ranks not available but Crunchbase Rank uses multiple signals.
  • Coverage
    • Has data for most SMBs and major startups (covers 25,000 orgs in India)
    • Private, innovative, growing companies
    • Data is updated atleast once a week.

Endpoints of interest

  • Searching for organisations by name or domain : /organizations

    import requests
url = "https://api.crunchbase.com/v3.1/organizations"
CB_TOKEN = "TOKEN"
querystring = {"domain_name":"freshworks.com","user_key":CB_TOKEN}
response = requests.get(url, params=querystring)
perma_link_id = response.json()['data']['items'][0]['properties']['permalink'] 

  • Getting properties of an organisation :
    Organisation information retrieved API is same as Crunchbase page without the 3rd Party Data (Owler, BuiltWith, etc.) 

    import requests

perma_link_id = "Freshworks"
url = "https://api.crunchbase.com/v3.1/organizations/"+perma_link_id
querystring = {"user_key":CB_TOKEN}
response = requests.get(url,params=querystring)

data = response.json()

Other Notes 

  • data['properties'][u'also_known_as'] - Company name aliases (Could help with dedupe, similar to domain aliases provided by clearbit)

Pricing

  • Flat fee for a 12-month or multi-year option for licensing their data. The flat fee is dependent on a variety of factors
  • Internal Use only:
    • Falls under Enterprise license. 
    • Allows us to use the data internally (only Freshworks employees are permitted to use and see the data). 
    • Baseline package is $12,000 USD for 12-months, and gives access for 5 developers and up to 25 viewers. 
    • Scales up from there depending on the number of view-only users. 
    • They also offer a truly unlimited package for unlimited users in the six figure range.
  • Commercial Use (exposing to customers)
    • Falls under Applications license. 
    • Flat fee is dependent on the data fields we want to expose to customers. 
    • All in package is usually $80,000-$90,000 USD, and there are some restrictions on this (no exporting, and we need to cite Crunchbase as a source of the data).
